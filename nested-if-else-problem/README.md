#  NESTED IF ELSE PROBLEM STATEMEMT
A tea shop offers discounts based on the number of tea cups ordered. Write a program that checks the number of cups ordered and applies a discrount. More than 20 cups 20% discount.

condition:
(01) => Between 10 and 20 cups: 10% discount.
(02) => Less than 10 cups: No discount
(03) => More than 20 cups: 20% discount.
